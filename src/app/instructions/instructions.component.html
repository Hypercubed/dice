Decode In Case of Emergency (DICE) is a simple online and open-source tool to convert confidential messages into encrypted QRCodes to be shared with your loved ones and decoded in case of emergency (or for any reason whatsoever).

Messages are encoded using an encryption pass phase and aes-256-cbc encryption, the same as the one used in the OpenSSL cli tool (tested with OpenSSL v1.1). All encryption and decryption happens within the browser and never sent to any server. Feel free to deploy your own site for this tool; and don't worry messages can always be decrypted using the OpenSSL CLI tool as well.

How to use Dice

Encoding

  1. Enter an encryption pass phase. This pass phase is shared with your intended recipient(s). This pass phase cannot be recovered.

  2. Enter text to encrypt. The text will be encrypted using the encryption pass phase you entered. The text is encrypted with aes-256-cbc encryption, the same as the one used in the OpenSSL cli tool.

  3. Download QR Code or encrypted text. This QR code can be embedded in a document, printed and stored in case of emergency. The QR code data includes the encrypted text as well as a link to this site.

Using OpenSSL: echo -n "{{secret message}}" | openssl enc -e -aes-256-cbc - -base64 -pbkdf2

Decoding

  1. Enter an encryption pass phase. This pass phase that was shared with you.

  2. Scan the QR code or enter the encrypted text. If you arrived at this site by scanning a QRcode this will be populated already. The text will be decrypted using the encryption pass phase you entered.

  3. Decrypted Text is displayed.

Using OpenSSL: echo -n "{{encrypted message}}" | openssl enc -d -aes-256-cbc -A -base64 -pbkdf2
